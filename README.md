# Day Comment

## Configuration

`"prefix": "######"`
The tag before moment.

`"format": "YYYY/MM/DD ddd HH:mm"`
Moment format.

`"list": ["今日计划", "今日记录", "明日计划"]`
Daily record.


## Shortcut

Default shortcut is `ctrl + f5`  / `cmd + f5`


## Example

```
###### 2019/01/09 Wed 22:20
- 今日计划

- 今日记录

- 明日计划

```


**Enjoy!**
